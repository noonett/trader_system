# 交易日志模板

> 诚实是一切的基础。对自己撒谎比亏钱更致命。

---

## 开仓记录

```
日期：     YYYY-MM-DD
时间：     HH:MM （北京时间）
标的：     [代码/名称]
方向：     [多 / 空]
仓位：     [股数/合约数]
入场价：   [价格]
账户总资产：[人民币/港币]

—— 决策链 Q1-Q5 答案 ——

Q1（论据）：
[你的答案]

Q2（止损条件）：
[具体价格点位或事件条件]

Q3（全亏影响）：
[生活影响 + 心理影响]

Q4（情绪状态）：
[情绪词] 强度：[1-5]

Q5（胜率/赔率估算）：
预期胜率：[%]
预期盈利：[金额/百分比]
预期亏损：[金额/百分比]
期望值：[计算结果]

Q5a（估算依据）：
[选择一项：实际数据 / 回测 / 直觉 / 他人建议 / 其他]

—— 决策链记录结束 ——
```

## 平仓记录

```
平仓日期：   YYYY-MM-DD
平仓时间：   HH:MM （北京时间）
平仓价：     [价格]
持仓天数：   [天数]
盈亏：       [+/- 金额]  ([+/- %])

结果分类：
[ ] 止盈
[ ] 止损
[ ] 手动平仓（非触发条件）

持仓期间情绪变化：
[如有剧烈波动，记录情绪演变过程]
```

---

## 事后复盘

```
1. 这笔交易按照当时的计划执行了吗？
   [ ] 完全按照计划
   [ ] 部分偏离（说明：______）
   [ ] 完全偏离（说明发生了什么）

2. 止损是否被执行？
   [ ] 是，到了止损点出场
   [ ] 否，未到止损点但手动出场
   [ ] 否，到了止损点但未执行（记录原因：______）

3. 仓位是否在合理范围内？
   [ ] 是
   [ ] 否（大了/小了，下次怎么调整？）

4. 这笔交易给你什么教训？
   [一句话总结]

5. 下次同类交易你会做什么不同？
   [具体改进点]
```

## 完整日志示例

```markdown
## 2026-05-03 000300.SH

### 开仓
- 时间：2026-05-03 10:30 CST
- 标的：沪深300 ETF (510300)
- 方向：多
- 仓位：1000 份
- 入场价：3.850
- 账户总资产：100,000 CNY

### 决策链
Q1: 沪深300在3.800附近获得支撑，30分钟线出现放量阳线，突破3.850阻力位。
Q2: 如果价格跌破3.800，支撑失效，止损出场。
Q3: 1000份 = 3,850元，占总资产3.85%。全亏不影响生活，但会影响心态。
Q4: 平静 强度：2
Q5: 预期胜率 60%，平均盈利 200元，平均亏损 150元。期望值 = 60%×200 - 40%×150 = 60元。
Q5a: 过去类似支撑位突破的交易做过几次，胜率大约在5-6成。

### 平仓
- 平仓日：2026-05-07
- 平仓价：3.920
- 盈亏：+70元 (+1.82%)
- 分类：止盈

### 复盘
1. 完全按计划执行。
2. 止损未触发。
3. 仓位合理。
4. 等待突破确认后再入场比提前埋伏更稳妥。
5. 下次可以在突破确认时适当加大仓位。
```
