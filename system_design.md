## 一、交易哲学

### 我们以什么框架理解市场？

**底层哲学：多层过滤框架 (Layered Filter Framework)**

市场有四个层次，从上到下逐级过滤信号。90% 的散户从 L2 开始，所以他们亏钱。

```
L0 宏观环境（时势）      "现在是什么世道？"
        ↓ 决定策略方向
L1 资金流向（热度）      "钱在往哪流？"
        ↓ 决定战场选择
L2 基本面（价值）        "什么东西值得买？"
        ↓ 决定标的池
L3 技术面（时机）        "什么时候动手？"
        ↓ 决定买卖点
```

**L0 为什么最重要？** 同一个公司的股票，在 2020 年（放水）和 2022 年（收水）估值可以差 3-5 倍。选再好的股票，宏观错了照样亏钱。

**对你来说意味着什么？** 每次想买一只股票之前，先问四个问题：
1. 现在 L0 是什么环境？应该积极还是保守？
2. L1 钱在不在这个板块？
3. L2 这家公司真的好吗？
4. L3 现在是好的买入时机吗？

### 我们的核心原则

**趋势大于估值。** 便宜的东西可以更便宜，贵的东西可以更贵。在趋势逆转之前，不急着抄底。

**流动性是第一推动力。** 央行比所有基本面都重要。流动性收紧时，现金就是最好的仓位。

**概率思维。** 没有"一定会涨"的股票，只有"概率更高的情境"。永远控制仓位。

**不做清单。** 知道自己不做什么比知道做什么更重要。这个清单会随着犯错次数增加而变长。

**复盘 > 预测。** 猜明天涨跌是扔硬币。回头看"今天发生了什么，为什么我的判断对了/错了，下次怎么改进"才是积累。

## 三、AI 大脑层：AI-native 的核心

这才是真正"AI 原生"的部分——不是代码自动化，而是**把每天的信息变成你能读懂的叙事**。

### 设计原则

**自然语言交互。** 你问"今天市场怎么样？"，AI 给你一段分析，不是表格。

**上下文记忆。** AI 记得昨天说了什么、上周关注了什么标的、你的持仓情况。

**可解释。** 每个结论都注明"我是基于什么信息和知识得出这个判断的"。

### AI Agent 角色

| Agent | 职责 | 频率 |
|-------|------|------|
| 盘前简报官 | 隔夜外盘 + 今日重要事件 + 值得关注的标的 | 每日 |
| 收盘复盘官 | 今天市场主线、资金干了什么、明天看什么 | 每日 |
| 宏观哨兵 | 利率、汇率、政策的重大变化 | 事件驱动 |
| 自选股管家 | 跟踪自选股池核心指标，异动提醒 | 持续 |
| 错误分析师 | 当你做错一笔，帮你解剖原因，记入日志 | 按需 |
| 学习导师 | 根据你的认知水平推荐该学的内容 | 每周 |

### 每日简报示例（这是我们想要的效果）

```
【2026-05-03 盘前简报】

今天关注三件事：

1. 流动性偏紧——央行本周净回笼 2000 亿，银行间利率小幅上行。
   含义：短期对高估值成长股不利，资金可能往价值股靠。

2. 黑色系期货夜盘全线下跌——螺纹钢跌 2%、铁矿石跌 3%。
   含义：可能反映对基建开工的担忧，今天留意 A 股钢铁、建材板块。

3. 北向资金连续 3 日净流入——周五再度净买入 50 亿，主要流入消费和新能源。
   含义：外资对中国资产偏正面，但还需看持续性。

提醒：你持仓的 XX 股份今晚发年报，关注毛利率是否持续下滑。
风险提示：下周美国 CPI 数据，可能引发全球波动，建议控制仓位。
```

## 五、迭代路线：怎么一起成长

```
Stage 0 — 地基 (本周)
├── 知识库第一版（哲学、术语、风控规则）
├── 你选标的 → 我做第一份深度分析
├── 每日简报框架
└── 工具层：数据获取 + 基础比率

Stage 1 — 认知起步 (第 2-4 周)
├── 每天一份盘前简报
├── 每天收盘后复盘
├── 建立自选股池（5-10 只，你选板块我分析）
├── 知识库补充：遇到的新概念都记下来
└── 工具层：DCF 估值 + 筛选器

Stage 2 — 独立判断 (第 2-3 个月)
├── 你能自己判断"今天最重要的事是什么"
├── 能做单票深度分析，AI 辅助验证
├── 错误日志积累 → 你自己的交易原则开始成形
└── 工具层：板块轮动分析 + 监控预警

Stage 3 — 系统化 (第 4-6 个月)
├── 有自己完整的交易系统
├── 从"跟着走"到"在理解的基础上做选择"
├── 知识从学到创——你开始形成自己的方法论
└── 工具层完全自动化，注意力回到思考本身
```

## 七、风险和边界

1. **这不是投资建议系统。** 所有分析都是"信息整理 + 概率判断"，最终决定永远是你自己做的。
2. **AI 会犯错。** 我的数据可能有延迟，我的分析逻辑可能有盲区。交叉验证永远是必要的。
3. **市场不可预测。** 任何人告诉你"这样一定赚钱"，不是无知就是骗子。
4. **学习有曲线。** 开始 1-2 个月会有很多不懂的概念，这完全正常。每个人都是从小白开始的。
