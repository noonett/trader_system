# σ 固定时刻提醒 — macOS / iOS Calendar 模板

> 对应 design_proposal_2026.md §三.2.4 + §二 D4。
> Mobile 仅承担时点提醒，不承担决策链 / 写入路径（§收敛信号 2）。

## 设置方法

在 macOS 日历 / iPhone 日历中创建以下 5 条**重复事件**。
建议新建一个专用日历（如"σ 交易提醒"），方便统一开关。

---

## 事件 1：盘前规则书写

| 字段 | 值 |
|---|---|
| **标题** | 📝 σ 盘前规则书写 |
| **时间** | 08:30（工作日） |
| **重复** | 每周一至周五 |
| **提醒** | 事件时 + 提前 5 分钟 |
| **备注** | 打开 Cursor → sigma/daily/YYYY-MM/YYYY-MM-DD-pre.md → 写 if-then 规则 |

## 事件 2：集合竞价倒计时（A 股）

| 字段 | 值 |
|---|---|
| **标题** | ⏰ 9:20 集合竞价不可撤单 |
| **时间** | 09:15（工作日） |
| **重复** | 每周一至周五 |
| **提醒** | 事件时 |
| **备注** | 打开当日 pre 文件 → 强制复述今日 if-then → 9:20 后不可撤单 |

## 事件 3：盘后 EMA 截止

| 字段 | 值 |
|---|---|
| **标题** | ✏️ σ 盘后 EMA（平仓后 30 分钟内） |
| **时间** | 15:30（工作日，A 股收盘后 30 分钟） |
| **重复** | 每周一至周五 |
| **提醒** | 事件时 |
| **备注** | 若今日有平仓 → 在 trades/ 对应文件填 §四 EMA ≤4 字段 → git commit |

> 如果你交易的是美股期货（如 MGC），把时间改为对应市场的收盘后 30 分钟。

## 事件 4：周复盘

| 字段 | 值 |
|---|---|
| **标题** | 📊 σ 周复盘 |
| **时间** | 周日 20:00 |
| **重复** | 每周日 |
| **提醒** | 事件时 + 提前 1 小时 |
| **备注** | 终端 → make weekly-report → 读 violations + auto 报告 → 写 reviews/weekly/ |

## 事件 5：月校准

| 字段 | 值 |
|---|---|
| **标题** | 📈 σ 月校准 |
| **时间** | 每月最后一个周日 20:00 |
| **重复** | 每月（手动调到最后一个交易周末） |
| **提醒** | 事件时 + 提前 1 小时 |
| **备注** | 终端 → make monthly-calibration → KPI 退化判定 → 写 reviews/monthly/ |

---

## 注意事项

- **不要**在手机上打开交易 App 执行下单——Calendar 仅做提醒，决策链在 Cursor 桌面完成
- 出差/假期：提前一天在 sigma/daily/ 写"模板规则"占位（design §二 D11 missed-day recovery）
- 连续 3 天未写规则：系统提示"考虑暂停 1 周复盘"——不强制 lockout
