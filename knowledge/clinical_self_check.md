# σ 系统 — N=1 临床安全自检模板

> **本模板的核心约束（v3 §6.1 N=1 原则）**：
>
> 1. **不替代专业诊疗** — 这是行为意识的自我监测工具，不是临床诊断。任一项触发都不构成 medical advice。
> 2. **不 lockout gate** — σ 系统不基于自评结果"拒绝你接入"。自评是为你**自己提供数据**，不是系统对你的判决（v2 §6.1）。
> 3. **用户自决何时重测** — 建议的频率是月度 + 情绪明显波动 / 重大亏损后；但具体由你决定。
> 4. **诚实措辞调整**（v3 §6.6 A23）：避免"赌博"用语（中国大陆道德污名 + 法律联想 → 降低填表诚实度）；用"投机交易行为"或"高风险下注行为"中性技术语。
> 5. **状态性风险防御保留** — 即便量表分数不在极端高风险，连续亏损 + 睡眠剥夺 + 重大生活事件等状态性条件仍触发"考虑暂停"提示（v3 §6.1 #3）。

## 一、入门基线（建议在 v0 启动时跑一次）

预计耗时：~5–10 分钟。

### 1.1 PHQ-9（抑郁筛查 + 含自杀题）

> **来源**：Kroenke, Spitzer & Williams (2001) *J Gen Intern Med* — Patient Health Questionnaire-9。
> **v3 升级原因**（§6.2）：v1 PHQ-4 不含自杀题，σ 目标人群（亏损 + 自责 + 男性 + 反刍）与高自杀风险人群结构性重叠 → 必须升级。

**最近 2 周内，下面 9 个问题困扰你的频率是？**

记分：0 = 没有 / 1 = 几天 / 2 = 一半以上的天数 / 3 = 几乎每天


| #   | 问题                             | 分数  |
| --- | ------------------------------ | --- |
| 1   | 做事时缺乏兴趣或乐趣                     | __  |
| 2   | 感觉沮丧、抑郁或绝望                     | __  |
| 3   | 入睡 / 维持睡眠困难，或睡得太多              | __  |
| 4   | 感到疲倦或精力不足                      | __  |
| 5   | 食欲不振或吃太多                       | __  |
| 6   | 觉得自己很糟糕——觉得自己是失败者，或让自己 / 家人失望  | __  |
| 7   | 难以集中注意力（如阅读 / 看电视）             | __  |
| 8   | 行动 / 言语缓慢到别人能注意到；或反过来焦躁不安、来回走动 | __  |
| 9   | **觉得自己活着不如死了好，或想以某种方式伤害自己**    | __  |


**总分**：__ / 27

**参考阈值**（仅供自我意识，非诊断）：

- 0–4：minimal（最小）
- 5–9：mild（轻度）
- 10–14：moderate（中度）
- 15–19：moderately severe（中重度）
- 20–27：severe（重度）

**触发"考虑暂停"提示**：

- **第 9 题任一分 ≥ 1**（任何频率的自杀 / 自伤念头）→ 立即停止使用本系统并寻求专业帮助；拨打 §四 的心理援助热线
- 总分 ≥ 15（中重度）→ 考虑暂停 σ 训练 + 就医评估
- 总分较上次基线骤升 ≥ 5 → 考虑暂停训练

### 1.3 5 项投机交易行为自查（A 股本地化措辞）

> **来源**：基于 DSM-5 problem gambling 9 项的 σ 系统简化版（v3 §6.6 A22 + §6.5）。
> **v3 措辞调整**：避免"赌博"用语，使用"投机交易行为 / 高风险下注行为"。

**最近 30 天内，以下情况对你成立吗？**


| #   | 描述                                    | 是 / 否 |
| --- | ------------------------------------- | ----- |
| 1   | 我曾尝试减少或停止交易，但失败了                      | __    |
| 2   | 不能交易时（周末 / 假期 / 持仓被锁），我感到焦躁、烦闷或注意力难集中 | __    |
| 3   | 亏损后，我通过加大头寸 / 频繁交易"追平"过               | __    |
| 4   | 我对家人 / 重要他人隐瞒过我的交易亏损或交易次数             | __    |
| 5   | 我感到无助 / 内疚 / 焦虑 / 抑郁时，会通过交易缓解情绪       | __    |


**触发"考虑暂停"提示**：

- 任 1 项触发 → 注意当前模式
- **#5 单独触发**（即使其他 4 项都未触发）→ **强烈建议暂停训练 + 寻求专业帮助** —— 这是 σ 系统**最危险的过渡机制**（v3 §6.3 distress-driven trading）
- 任 2 项及以上触发 → 强烈建议暂停训练 + 评估

### 1.4 DSM-5 problem gambling 9 项（按 σ 语境改写，月度自查）

> **来源**：DSM-5 (American Psychiatric Association 2013) Gambling Disorder criteria，按 σ 系统语境改写（v3 §6.3）。
> **诊断阈值**：DSM-5 临床诊断要求最近 12 月内任 4 项触发——**不是诊断**，是自我意识 checklist。

**最近 12 月内**：


| #   | 项目          | 描述                            | 是 / 否 |
| --- | ----------- | ----------------------------- | ----- |
| 1   | 耐受          | 需要逐渐放大头寸 / 杠杆才能获得相同的兴奋或刺激     | __    |
| 2   | 戒断样反应       | 周末 / 假期 / 不能交易时焦躁、烦闷、注意力难集中   | __    |
| 3   | 反复尝试停止失败    | 曾多次尝试减少 / 停止交易但失败             | __    |
| 4   | 心智占据        | 大量时间用于规划下次交易 / 复盘上次交易 / 想方法翻本 | __    |
| 5   | **痛苦驱动型交易** | 感到无助 / 内疚 / 焦虑 / 抑郁时通过交易缓解情绪  | __    |
| 6   | 损失追逐        | 亏损后通过加大下注 / 频繁交易"追平"          | __    |
| 7   | 撒谎掩盖        | 对家人 / 重要他人撒谎以掩盖交易程度           | __    |
| 8   | 关系/工作/学业损害  | 因交易失去重要关系 / 工作 / 学业机会         | __    |
| 9   | 依赖他人金钱      | 因交易导致的财务困境而依赖他人提供金钱           | __    |


**触发提示**：

- **任 4 项触发**（DSM-5 临床阈值）→ **强烈建议暂停训练 + 寻求专业帮助**
- **#5 痛苦驱动型交易单独触发**（即使其他 8 项都未触发）→ **强烈建议暂停 + 寻求专业帮助**（σ 系统最危险过渡机制，v3 §6.3）

### 1.5 5 项简化绝对禁忌症自查（v3 §6.5）

> **N=1 自用语境**：5 项最关键自查；触发时仅作"如自查触发请暂停"提示（不强制 lockout）。

**如果你当前满足以下任一条，强烈建议暂停使用本系统并寻求专业帮助**：

- 当前或近 6 月有重度抑郁发作（与 §1.1 PHQ-9 总分 ≥ 20 互相印证）
- 当前有自伤意念，或最近 12 月内有过自杀未遂（与 §1.1 PHQ-9 第 9 题互相印证）
- 你正在借钱交易 / 隐瞒家人交易 / 亏损后频繁加码"翻本"
- 交易已经显著影响你的睡眠 / 工作 / 重要关系
- 你感到无法停止看盘 / 反复进入交易工具 / 无交易日严重焦躁

**productization 路径完整版**（参见 [pr_3_review_04_clinical_claude_first_pass.md](../research/pr_3_review_04_clinical_claude_first_pass.md) "漏掉的绝对禁忌症 Tier A + Tier B 完整清单"）：~15 项，包括活跃精神病性症状、未控 ADHD、活跃 SUD、PTSD 急性、解离活跃、法律债务诉讼、重大丧失 < 3 月、多巴胺激动剂用药等。N=1 自用语境只列 5 项最关键的。

## 二、状态性风险触发（v3 §6.1 #3，不依赖特质风险等级）

**即便上面所有量表都未触发，以下状态性条件仍应触发"考虑暂停"提示**：

- 最近 3 天连续亏损 + 睡眠不足 6 小时
- 最近 30 天发生重大生活事件（搬家 / 失业 / 关系危机 / 健康事件 / 家人病逝）
- 心理痛苦自评（0–10）≥ 7
- 连续 7 天感到"明显比平时焦虑 / 易怒 / 失眠"

**任 1 项触发** → 考虑当周降低交易频率 / 缩小仓位 / 暂停训练。

## 三、诚实措辞约束（v3 §6.1 #4，所有语境通用）

> 这一节不是临床要求，是认知校准（foundation §三.2 反馈组件角色约束 + v3 §6.1 #4 诚实措辞临床调整）。

**审查的是记录质量，不审判人格**：

- ❌ 旧版（容易诱发自我攻击）："你今天诚实吗？"
- ✅ 新版（审查记录质量）："这条记录是否足够可检验？"

**对应的自检措辞**：


| 自检场景    | 旧措辞（避免）            | 新措辞（推荐）                                 |
| ------- | ------------------ | --------------------------------------- |
| 决策链 5 问 | "你诚实地回答了 Q1-Q5 吗？" | "Q1-Q5 的答案是否包含可被未来自己 fact-check 的具体内容？" |
| 周复盘     | "你这周诚实记录了吗？"       | "这周的记录条目是否可以与券商交割单 + git log 对账？"       |
| 月校准     | "你诚实评估自己了吗？"       | "事前预设 vs 实际发生 vs 偏差三段是否完整？"             |


**理由**（CBT / ACT 自我同情 vs 自我批判文献，M/S）：人格判定型措辞激活防御性反应 → 降低填表诚实度；记录质量审查型措辞激活元认知 → 提升填表诚实度。

## 五、使用流程（N=1）

```
v0 启动时：
  不阻塞——直接写第一份盘前规则即可。
  基线自检不是 v0 启动条件。

v0 启动后首月内（建议，不强制）：
  1. 找一个非交易时段，跑一次完整 §一（~10 分钟）
  2. 把结果存到 memory/clinical_baseline.md
  3. 任一节触发"强烈建议暂停" → 考虑是否继续 σ 训练
  4. 全部未触发 → 这份基线留着，以后月度 / 重大事件后对比用

之后：
  - 月度（建议每月最后一个交易日盘后）：跑一次 §一 各节
  - 情绪明显波动 / 重大亏损 / 重大生活事件后：跑一次 §一
  - 每个交易日盘前：用户主观判断是否需要跑 §二 状态性风险检查（不强制）

任一时刻触发"考虑暂停"提示：
  → 先暂停当周 / 当日交易（不强制 lockout，由你决定）
  → 重启时建议至少完成 §一 + §二 复评
  → 第 9 题 PHQ-9 ≥ 1 / 5 项禁忌症 #2 触发 → 拨打 §四 资源 + 暂停 σ 训练
```

## 七、来源 / 诚实标记

**S 级来源**：

- Kroenke, Spitzer & Williams (2001) *J Gen Intern Med* — PHQ-9
- Spitzer et al. (2006) *Arch Intern Med* — GAD-7
- DSM-5 (American Psychiatric Association 2013) — Gambling Disorder

**M 级来源**：

- Phase 2 调研 [research/notes/12_no_ai_option.md](../research/notes/12_no_ai_option.md) §2.1（OCD/反刍/高焦虑/完美主义/抑郁严重用户对 AI 前台的 5 类已识别反向机制）
- 中国大陆心理援助资源（电话号码基于 2026 Q2 训练数据，需人工 re-verify）

**U 级（明确标注）**：

- 5 项投机交易行为自查的具体措辞调整（来自 DSM-5 9 项的 σ 系统改写，未经独立 RCT 验证 A 股语境的有效性）
- 状态性风险触发条件（连续亏损 + 睡眠剥夺 + 重大生活事件 + 心理痛苦自评 ≥ 7）的具体阈值
- "诚实措辞调整"（§三）对填表诚实度的影响是基于 CBT / ACT 文献的逻辑推演，没有 σ 语境的直接 RCT

**整体诚实标记**：本模板**不是临床诊断工具**。N=1 自检的核心目的是**给你自己提供数据 + 触发停一停的提醒**，不是替代专业诊疗。如果任一节触发"强烈建议暂停"或"寻求专业帮助"——**请认真对待**，不要因为这是"自评"就忽略它。
