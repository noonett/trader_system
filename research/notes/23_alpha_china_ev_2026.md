## 五、海外市场：欧盟价格承诺 + 新兴市场

### 5.1 欧盟"以价格承诺替代关税"（2026-01-12 落地）

**原关税情况**（2024-10 宣布）：
| 企业 | 反补贴税率 | + 10% 标准关税 |
|---|---|---|
| 比亚迪 | 17% | 27% |
| 吉利 | 18.8% | 28.8% |
| 上汽 | 35.3% | 45.3% |
| 特斯拉中国 | 7.8% | 17.8% |

**2026-01-12 新机制**：欧盟委员会发布《价格承诺申请指导文件》，中欧双方达成"价格承诺"机制替代高额关税。

**核心要点**：
- 企业承诺**最低进口价格（MIP）** → 避免高额关税
- MIP 设定可选：(a) 出口商调查期到岸价 + 反补贴税额；或 (b) 欧盟非补贴同类销售价 + 合理利润
- 投资承诺非强制但有利于通过申请
- 可单独或联合申请

来源：[商务部官方解读](http://chinawto.mofcom.gov.cn/article/dh/cyjieshao/202603/20260303621867.shtml)、[证券时报 2026 解读](https://www.stcn.com/article/detail/3593791.html)（M）。

**关键解读**：
- **这是真利好**——比 35% 关税显著缓和
- **但 MIP 机制意味着中国车在欧洲定价被锁定**——量增但单价被压制
- **欧洲电动车业务真实可持续**，但**利润空间被欧盟分走一部分**

## 八、与 σ 风控规则的对账 + 子赛道分级

### 8.1 不能笼统说"电动车板块好"

按 σ 元规则要求，**本笔记的核心结论之一**：电动车板块**不是一个整体可投资对象**——必须分子赛道、分公司层级判断。

### 8.2 σ 系统下的电动车子赛道分级

| 子赛道 | σ 评级 | 理由 |
|---|---|---|
| **电池龙头**（宁德时代） | 🟢 绿区+（合理仓位）| 持续盈利、市占率高、海外扩张、产业链定价权 |
| **整车头部海外业务**（比亚迪海外）| 🟢 绿区（合理仓位）| 真实增长、单车利润高 |
| **整车头部综合**（比亚迪整体）| 🟡 黄区（谨慎仓位）| 国内业务承压稀释海外亮点 |
| **新势力强势者**（蔚来、理想） | 🟡 黄区（谨慎仓位）| 销量增但盈利能力未验证、波动大 |
| **新势力调整者**（小米、小鹏） | 🔴 谨慎（更高门禁）| 销量分化、估值不确定 |
| **锂电二线**（亿纬除外） | 🔴 谨慎 | 净利 -70%/-80%，多数承压 |
| **传统车企转型** | 🔴 谨慎 | 转型阵痛、本笔记未深入 |
| **二三线整车 / 90% 退出名单** | 🔴 红区（拒入）| 倒闭风险 |
| **经销商 / 4S 店相关** | 🔴 红区（拒入）| 系统性崩盘 |

### 8.3 适用的 σ 风控规则

| 规则 | 在电动车板块的应用 |
|---|---|
| risk_rules §一 单股 ≤ 20% | 即使重仓宁德/比亚迪也不超 20% |
| risk_rules §一 同板块 ≤ 30% | 电动车 + 锂电 + 上游有色 + 半导体（智驾） **合计** ≤ 30% |
| risk_rules §二 止损 | 比亚迪逻辑止损例：H2 海外销量 < 月均 13 万 / 单车利润 < 8,000 元 |
| risk_rules §五 黑天鹅 | 月度乘联会数据日 + 公司业绩日 + 欧盟 MIP 谈判节点 → 不开仓窗口 |
| risk_rules §七 危险信号 | "电动车板块好" → "梭哈" 是变体；本笔记的分子赛道判断必须用 |
| risk_rules §九.0 资金分离 | 宁德 / 比亚迪 H 股可作为**长期财富资金**的科技板块配置 |

### 8.4 如果你坚持要在电动车板块配置，最小门禁

> **不是推荐你买，是"如果你已决定，最低门禁"**

1. **优先级排序**：宁德时代 > 比亚迪海外 > 新势力强者 > 锂电亿纬 — 不在"二线整车 / 二线锂电 / 经销商" 浪费仓位
2. **仓位上限**：单股 ≤ 15-20%；电动车 + 上游合计 ≤ 30%
3. **入场点**：参考 Forward PE < 18× 区间（比亚迪 90-105 港元）
4. **决策链 Q2 被推翻条件示例**：
   - 比亚迪：海外月销 < 13 万、单车利润 < 8,000 元、Q2 净利 < 50 亿
   - 宁德：动力市占率回到 40% 以下、海外份额停滞
5. **不在以下场景买入**：
   - 看到"出口创新高" 新闻当天追涨
   - 板块单周涨 > 15%
   - 听到"电动车肯定还会涨" 的群推荐

## 九、综合判断与可行的下一步

### 9.1 三句话总结

1. **"电动车板块基本面真实可持续" 是过度简化**——板块内的真实可持续仅集中在 **电池龙头（宁德）+ 整车头部海外业务**，国内整车 + 二三线 + 经销商正在系统性出清
2. **板块利润高度集中**——宁德一家 ≈ 15 家车企累计；定价权已从整车向上游转移
3. **当前估值合理**（比亚迪 H 股 Forward PE 18.5×、PEG 0.70）但隐含假设较激进（海外销量 +120% + 单车利润维持 1.1 万）

### 9.2 可行的看法

**看法 A：聚焦电池+整车海外双龙头**（**最干净**）
- 工具：宁德时代 H（3750.HK） + 比亚迪 H（1211.HK）
- 仓位：单股 ≤ 15%、合计 ≤ 25%
- 假设：宁德电池统治地位 + 比亚迪海外执行力
- 被推翻条件：见 §8.4

**看法 B：板块 ETF**
- 工具：港股新能源车 ETF / 中证新能车 ETF（具体代码自查）
- 优势：分散单股风险
- 劣势：稀释赢家利润、被二三线拖累

**看法 C：仅配置宁德时代单一**
- 这是最集中赌"上游定价权转移" 的做法
- 仓位 ≤ 15%
- 风险：技术路径变化（固态/钠电）+ 上游资源价波动

**看法 D：不参与 / 等更便宜入场**
- 等待信号：比亚迪股价 < 90 港元（强吸引力区）/ 国内整车价格战缓和 / 经销商网络企稳
- **完全合理**
