## 🚨 三件最紧迫的事（今晚 / 明天最早时间）

1. **去药店买维生素 B1（10 mg×100 片，1–2 元）+ 复合维生素 B**，明天起 **B1 100 mg/日**开始吃 —— 这是当前处方循证缺漏，OTC 安全
2. **启动家庭症状监测日记**（心率、食欲、呕吐、大便颜色每天记）
3. **本周做一次互联网血液科主任级问诊**（丁香医生 / 好大夫，附 4 张化验单照片）—— 性价比最高的二次意见

## 🚨 红线症状 → 立即急诊

任何一项出现**立刻去最大医院急诊，不要等下次复诊**：

- 黑便 / 呕血 / 柏油样大便
- 行走不稳 / 眼球抖动 / 意识模糊（Wernicke 脑病）
- 手脚麻木明显加重（B12 神经损害）
- 静息心率持续 > 110 + 呼吸急促
- 持续呕吐 24h 不能进食水
- 发热 > 38.5℃（白细胞低，感染高危）

详见文档 02。

## 📌 一段话总结

> **钟女士的"巨幼细胞性贫血"不是孤立急性病，而是"H. pylori 根除失败 → 慢性萎缩性胃炎 → 多种营养素吸收障碍 + 长期米粥饮食"的第二次更严重发作；当前处方（叶酸+甲钴胺+生血宁）只覆盖了急性血液学纠正这一层，未覆盖三个循证关键点：(1) 长期米食人群必须先补的维生素 B1；(2) 治疗第一周必须密切监测的钾/磷/镁电解质崩溃风险；(3) 必须解决的根因——H. pylori 复查与二线根除、胃镜+OLGA 分期、可能的终身 B12 维持治疗。既往那次住院最大失误就是只做了第 1 层，所以才会复发并升级；这次必须借机把根因和长期管理一并纳入，否则循环还会第三次发生且更严重。**
