## 🚨 今晚 / 明天就做（家人自主，零风险，循证支持）

### 1. 加吃维生素 B1（硫胺素）—— 当前处方最重要的循证缺漏

**做法**：
- 任何药店或网上买**维生素 B1 片 10 mg/片**（OTC 非处方药，1–2 元一瓶，不需医生开）
- **剂量：100 mg/日**（10 mg × 10 片，分 2–3 次随餐吃），连续 **7–10 天**
- 之后转维持剂量 **10–20 mg/日长期**
- 同时买 **复合维生素 B 片**（含 B1/B2/B6/PP/B12），1 片每日 3 次

**为什么必须**（S 级，NICE 再喂养指南；ASPEN 2020）：
- 长期吃白米粥 → B1 严重缺乏（精白米几乎不含 B1）
- 启动 B12/叶酸 + 恢复进食时大量消耗 B1
- 缺 B1 → Wernicke 脑病（意识混乱、走路不稳、眼震）→ 不可逆，可致命
- B1 是水溶性维生素，过量从尿排出，**安全边界极宽**

**怎么和医生说**：下次复诊告诉医生"考虑到长期米粥饮食有 B1 缺乏风险，已经家里加吃 B1 100 mg/日"——医生通常会认可。

### 2. 启动渐进恢复进食方案（不要突然吃多）

**前 3 天**（关键）：
- 每日热卡 50% 起步（约 700–800 kcal）
- 蒸蛋羹 1 个 + 鱼蓉粥 1 碗 + 豆腐 100 g + 瘦肉末粥 1 碗
- 少食多餐 5–6 餐/日
- ⚠️ **避免**：突然喝大量糖水/葡萄糖、一次吃很多米饭（会触发 Wernicke）

**第 4–7 天**：加蔬菜泥（深色蔬菜：菠菜、油菜——叶酸+铁）

**第 7 天后**：引入猪肝（每周 2–3 次，B12+铁）+ 全谷物粥 + 深绿叶蔬菜

**永久戒掉**："只喝白米粥"的模式

### 3. 启动家庭症状监测日记（手机备忘录或纸笔）

**每天记录**：
| 项目 | 怎么测 | 警戒线 |
|---|---|---|
| 静息心率 | 早晨醒来躺床上数 1 分钟 | 持续 > 100 = 联系医生<br>持续 > 110 = 急诊 |
| 食欲 / 进食量 | 每餐吃了多少（比例） | 持续 < 50% = 联系医生 |
| 呕吐 | 频率、量、有无血丝 | 任何带血丝/咖啡渣 = **急诊** |
| 大便颜色 | 每次记录 | **黑便/柏油样/血便 = 立即急诊** |

**每 3 天测**：体重（同时间同衣服）、血压（如有血压计）

## 📞 明天 / 后天做

### 4. 联系开处方的医生（电话 / 加号），主动提 5 个问题

不要等下次复诊。打电话或挂号，直接问：

1. "已经在家加了维生素 B1 100 mg/日和复合 B，请问医生是否同意？"
2. "她的血常规、电解质、网织红、铁三项、TSH、B12/叶酸定量、抗内因子抗体——这些什么时候安排查？"
3. "甲钴胺现在是口服的，能否考虑先肌注 500 μg/日 × 2 周？她有慢性胃病，担心吸收不好。"
4. "治疗第 3 天和第 7 天我们应该来复查电解质（钾、磷、镁）和网织红吗？"
5. "她以前 H. pylori 治疗没成功，这次贫血稳定后多久可以做胃镜？"

📌 **关键**：**把问题写下来或打印**带去医院。准备过、有问题清单的家属医生通常会更认真。

## 📅 本周做（重要！）

### 5. 一次互联网血液科主任级问诊（性价比最高的二次意见）

**平台推荐**：丁香医生、好大夫在线、京东健康

**操作**：
- 找**血液科主任医师**（认准"主任医师"四个字）
- 上传 4 张化验单照片 + 处方笺照片 + 病史摘要（参考文档 01 患者画像段）
- 直接问 3 个问题：
  1. "目前这个处方够吗？是否需要补充什么？"
  2. "需要补哪些检查？什么时候做？"
  3. "目前严重程度是否需要住院？"

**费用**：200–500 元一次。**比错过早期胃癌或漏诊 MDS 的代价低数量级**。

## 🚨 红线症状 → 立即急诊（不要犹豫）

| 症状 | 临床含义 |
|---|---|
| **黑便 / 呕血 / 柏油样** | 上消化道出血，胃癌/溃疡需排除 |
| **意识模糊、走路不稳、眼球抖动** | Wernicke 脑病——必须急诊静脉补 B1 |
| **手脚明显麻木 + 行走异常** | B12 缺乏致脊髓亚急性联合变性 |
| **心率持续 > 110 + 呼吸急促 + 不能平卧** | 严重贫血/湿性脚气病心衰 |
| **持续呕吐 24h 不能进食水** | 脱水+电解质紊乱 |
| **发热 > 38.5℃** | WBC 2.85，感染高危 |
| **晕厥/跌倒/突发剧烈胸痛** | 紧急 |

**急诊去哪**：就近最大医院急诊。
