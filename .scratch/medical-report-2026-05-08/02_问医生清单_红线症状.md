# 02 问医生清单 + 红线症状 + 监测节奏

> **本文档**：13 条问医生清单（按优先级） + 红旗症状 + 治疗反应监测节奏
>
> **用法**：打印一份带去复诊；红线症状给所有家人都看一遍

---

## 📋 13 条问医生清单（按优先级）

### 🚨 紧迫层（24–48 小时内问）

**1. 维生素 B1（硫胺素）**

> "考虑到病人长期喝粥（精白米饮食）+ NICE 再喂养指南把她列为高危，是否需要在叶酸 / 甲钴胺之外加维生素 B1 100 mg/日？特别是启动造血和恢复进食前？"

**2. 甲钴胺给药途径**

> "甲钴胺现在是口服 0.5 mg×3/日，已知她有慢性胃病、H. pylori 治疗失败史，担心吸收不好。是否考虑先肌注 500 μg/日 × 2 周，再转口服？"

**3. CK-MB 异常解读**

> "5 月 8 日的 CK-MB 34 大于总 CK 24，这种数据怎么解读？是否需要查肌钙蛋白和心电图？是否考虑 macro-CK 检测？"

### 🔬 这周内的关键检查

**4. 病因学化验**

> "她的血清 B12、叶酸、同型半胱氨酸（HCY）、甲基丙二酸（MMA）这些病因学定量检查是否安排了？什么时候出结果？"

**5. 自身免疫 / 共病筛查**

> "抗内因子抗体（IFA）、抗壁细胞抗体（PCA）、TSH、FT4 是否在已开的检查里？因为老年女性恶性贫血常见，且可能合并自身免疫性甲状腺病。"

**6. 网织红细胞**

> "网织红细胞基线值是多少？治疗第 7 天复查吗？这是判断治疗反应最直接的指标。"

**7. 铁三项**

> "血清铁 / 铁蛋白 / 转铁蛋白饱和度查了吗？决定生血宁是否合理，也排除合并缺铁。"

**8. 电解质监测节奏**

> "治疗第 3 天和第 7 天我们应该来复查血钾、血磷、血镁吗？她 3 月份 K 已经 3.27 偏低，担心大量造血消耗电解质。"

### 🔍 1–2 周内（贫血稳定后）

**9. 胃镜 + 呼气试验**

> "她以前 H. pylori 治疗没成功，这次贫血稳定后多久可以安排胃镜 + 多点活检 + 13C 呼气试验？也想做 OLGA/OLGIM 分期决定监测频率。"

**10. 既往用药回顾**

> "她之前 H. pylori 根除用过哪几种抗生素？为了避免重复，下次二线方案需要避开。"

**11. 二线根除新方案**

> "如果检测出 H. pylori 仍阳性，二线根除是否考虑用伏诺拉生（vonoprazan, P-CAB）替代 PPI？是否联合益生菌？"

**12. 骨髓穿刺触发条件**

> "如果服药 7 天后网织红没升高、14 天后 LDH 没显著下降，是否安排骨髓穿刺排除 MDS？"

### 📅 长期管理

**13. 终身维持治疗**

> "如果证实恶性贫血或自身免疫性胃炎，是否需要终身肌注维生素 B12（每月 100 μg）？同时是否每 3 年胃镜监测胃癌？"

## 🚨 红线症状清单（任何一项 → 立即急诊）

⚠️ **这份清单建议家里多人都看一遍，避免漏判**

| 症状 | 提示 | 立即行动 |
|---|---|---|
| **黑便 / 柏油样大便 / 呕血 / 咖啡渣样呕吐** | 上消化道出血——**必须排除胃癌、胃溃疡** | 立即急诊 |
| **意识模糊、记忆突然变差、行走不稳、眼球抖动** | **Wernicke 脑病**（B1 缺乏，不可逆） | 立即急诊（要求静脉补 B1） |
| **手脚麻木明显加重 + 走路不稳** | B12 缺乏致脊髓亚急性联合变性 | 立即急诊 |
| **静息心率持续 > 110 + 呼吸急促 + 不能平卧 + 双下肢明显水肿** | 严重贫血 / 湿性脚气病心衰 | 立即急诊 |
| **持续呕吐 24h 不能进食水** | 脱水 + 电解质紊乱 | 急诊 |
| **发热 > 38.5℃** | WBC 2.85，**感染易致脓毒症** | 急诊 |
| **晕厥 / 跌倒 / 突发剧烈胸痛** | 严重贫血并发症 / 心血管事件 | 立即急诊 |
| **抽搐 / 严重肌无力 / 心悸严重** | 低钾 / 低磷 / 低镁 | 急诊 |

**急诊去哪**：
- 就近最大医院急诊（夜间 / 突发情况）
- 优先广东省人民医院河源医院（病历可调阅）
- 严重情况：广州省人医本部、中山大学附属医院

## 📊 治疗反应监测节奏（带去给医生确认）

| 时点 | 必查项目 | 关键指标 | 解读 |
|---|---|---|---|
| **基线（治疗启动前 / 当天）** | 网织红、外周涂片、B12、叶酸、HCY、MMA、抗 IF/PCA、铁三项、TSH、肌钙蛋白 | 网织红基线值 | 后续判断的对照 |
| **第 3 天** | 血钾、血磷、血镁、血钙 | K > 3.5、P > 0.7、Mg 正常 | 再喂养电解质监测 |
| **第 7 天** | 血常规、网织红细胞、电解质 | **网织红应跳升至基线 2–3 倍**（M 级） | **网织红不升 → 警惕 MDS / 溶血 / 肿瘤** |
| **第 14 天** | 血常规、**LDH**、电解质、AST | **LDH 应大幅回落**（如降至 < 1500） | **LDH 不显著下降 → 必须骨穿** |
| **第 14–21 天**（贫血稳定后） | 安排**胃镜 + 13C 呼气试验** | OLGA/OLGIM 分期 | 确定根因 + 监测策略 |
| **第 4–8 周** | 血常规、MCV、LDH | **Hb 大体回升正常**，MCV 开始下降 | 治疗诊断成立 |
| **3 个月** | 评估病因学 + 全身评估 | 是否需要终身维持 | 决定长期方案 |
| **长期** | 按 OLGA 分期定胃镜频率；每年复查血常规 + 铁三项 + B12 | — | — |

## 🩺 简单的"治疗成功"判断（家属也能看懂）

如果治疗确实命中诊断：

✅ **第 1 周内**：食欲好转、精神状态好转
✅ **第 2 周**：复查血常规网织红明显升高，LDH 大幅下降
✅ **第 4 周**：复查 Hb 至少升 10–20 g/L
✅ **第 8 周**：Hb 接近 100 g/L 以上，MCV 开始下降

如果出现以下情况 → **诊断可能错误 / 合并 MDS / 合并肿瘤 / 治疗不充分**：

❌ 第 7 天网织红没升高
❌ 第 14 天 LDH 没显著下降
❌ 第 4 周 Hb 没回升或继续下降
❌ 出现新的红线症状

**任一情况 → 推动骨穿 + 系统再评估**。

## 📝 把问题清单变成"医生面前能用"的工具

**实操技巧**：

1. **打印**这份清单，**带去医院**——不要凭记忆问
2. 进诊室前**只挑 3–5 个最重要的问题**，避免医生时间不够
3. 进诊室后**先说 1 句话总结**："医生，我们想确认 5 件事，每件 30 秒：……"
4. **带笔和小本子**记下医生回答（避免出门忘）
5. 如果医生时间不够，**主动说**："医生，剩下的问题我下次复诊再问，能否只回答最紧迫的 3 个？"——这种态度通常会让医生愿意多花点时间

**心理建设**：
- 医生面对"准备过、有问题清单、知道自己在问什么"的家属，**通常会更认真**
- 你不是在质疑医生，是在做合格的患者代理人
- 中国就诊"3 分钟门诊"是系统问题，**家属准备工作要弥补这个缺口**
