# 03 家庭行动方案

> **本文档**：今晚/这周/长期分阶段行动 + 饮食方案 + 家庭分工 + 症状日记模板
>
> **σ 元规则声明**：OTC 营养补充和饮食方案有强循证支持且安全；处方药改动、肌注、住院、转诊必须通过医生

---

## 🏠 第一类：今晚 / 明天就能做（家人自主，零风险）

### 1️⃣ 自购维生素 B1 + 复合 B（最重要）

**做法**：
- 任何药店 / 网上 / 医院药房买**维生素 B1 片 10 mg/片**（OTC，1–2 元一瓶）
- **维生素 B1 100 mg/日** = 10 片，分 2–3 次随餐吃
- 同时买**复合维生素 B 片**，1 片每日 3 次
- 连续 ≥ 7–10 天，之后转维持剂量 B1 10–20 mg/日长期

**循证依据**（S 级，NICE / ASPEN 2020）：
- 长期米粥饮食 + NICE 高危再喂养 → 必须先补 B1
- 不补 B1 + 启动造血/恢复进食 → Wernicke 脑病风险
- B1 是水溶性维生素，安全边界极宽，过量从尿排出
- **OTC 非处方药，不需医生开**

**告知医生**：下次复诊告诉医生"已加 B1 100 mg/日和复合 B"——医生通常会认可。

### 2️⃣ 维生素 C（100–200 mg/日）

促进非血红素铁吸收。OTC 安全，吃饭时一起吃。

### 3️⃣ ⚠️ 不要自行加铁剂

铁剂可能加重食欲不振 / 呕吐 / 便秘，且若不缺铁加用无益。**等铁三项结果出来再决定**。

## 🍲 NICE 再喂养指南渐进式恢复进食方案（家庭执行版）

### 阶段一：第 1–3 天（关键，必须慢）

**目标**：每日热卡按需求 50% 起步（女性约 700–800 kcal）

**示例餐单**：

| 时间 | 食物 |
|---|---|
| 早餐 | 蒸蛋羹 1 个（约 1 个鸡蛋）+ 小米粥半碗 + 豆腐脑半碗 |
| 上午加餐 | 苹果泥 半个 + 牛奶 100 ml |
| 午餐 | 鱼蓉粥 1 碗（鱼肉 30g）+ 蒸南瓜泥 |
| 下午加餐 | 蒸鸡蛋羹 半个 |
| 晚餐 | 瘦肉末粥 1 碗（瘦肉 30g）+ 嫩豆腐 |
| 睡前 | 温牛奶 100 ml（如能耐受）|

**禁忌**：
- ❌ 突然喝大量糖水 / 葡萄糖 / 果汁
- ❌ 一次吃很多米饭 / 面条
- ❌ 浓汤、油腻、辛辣
- ❌ 浓茶、咖啡（影响铁吸收）

### 阶段二：第 4–7 天

**目标**：热卡 75%，加深色蔬菜

**新增**：
- 菠菜泥、油菜泥、紫甘蓝泥（叶酸 + 铁 + 维生素 C）
- 蛋白量提升至每日 1 个鸡蛋 + 100g 鱼/瘦肉
- 可加小米/燕麦粥（B1 来源）

### 阶段三：第 7 天后

**目标**：热卡 100%，引入完整营养结构

**关键食材**（每周/每天）：

| 食材 | 营养 | 频率 |
|---|---|---|
| 猪肝 / 牛肝（小份）| **B12 + 铁 + B 族** | 每周 2–3 次（每次 50g）|
| 红肉（瘦牛肉、瘦猪肉）| **B12 + 铁 + 蛋白质** | 每天 50–100g |
| 鸡蛋 | B12 + 蛋白质 | **每天 1–2 个** |
| 深绿叶菜 | **叶酸 + 维生素 K** | **每天 1–2 份** |
| 全谷物（小米、燕麦、糙米）| **B1 + 膳食纤维** | 每天 |
| 豆制品 | 蛋白质 + 钙 | 每天 |
| 牛奶 / 酸奶 | 钙 + B12 | 每天 |
| 海产品（鱼类）| B12 + 蛋白质 | 每周 2–3 次 |
| 柑橘类水果 | 维生素 C | 每天 |

### 永久戒掉

❌ "只喝白米粥"的模式
❌ 三餐结构简陋（如只有粥+咸菜）
❌ 长期素食（B12 必须有动物性来源）

## 👨‍👩‍👧 家庭分工建议（慢性病程必须分工）

| 角色 | 负责 |
|---|---|
| **主照顾者**（配偶 / 同住子女）| 三餐、按时给药、夜间观察、症状日记日常记录 |
| **协调者**（学习能力较强的子女）| 与医生沟通、收集化验单、推动二诊、互联网问诊、整理资料 |
| **后备**（其他家属）| 急诊时人手、陪夜、情绪支持、采购药品食材 |
| **患者本人** | 配合按时吃药 + **如实告诉家人感受**（不要忍）+ 不私自加减药 |

## 💰 经济与资源现实考量

诚实标记：以下是 W 级（一般社会经验）+ U 级推演

**家人需要明白**：

1. **巨幼贫如果诊断正确，治疗反应通常很好**（M 级）—— 4–8 周 Hb 通常基本恢复
2. **真正的长期威胁是根因没解决**——胃癌风险、循环再发
3. **二诊（特别是上级医院 / 互联网血液专家）的钱不能省**——一次专家问诊 200–1000 元，相比错过早期胃癌或漏诊 MDS 的代价是数量级最高的性价比
4. **如果本院持续不给系统检查、不给住院，可能不是医生水平问题，而是地市医院资源分配问题**——这是患者权益问题，不是道德问题

**家人能掌控的事**：
- ✅ 营养与饮食（家庭执行）
- ✅ 症状监测（家庭执行）
- ✅ B1 / 复合 B / 维 C（OTC 自助）
- ✅ 互联网问诊（自费小钱）
- ✅ 推动当地医生加查
- ✅ 决定是否转诊上级医院

**家人不能掌控的事**：
- ❌ 床位资源（医院系统）
- ❌ 医生时间分配（系统约束）
- ❌ 医保覆盖范围（政策）

把精力放在能掌控的事上。

## 🌟 心理建设（给家人）

照顾慢性病人是长跑。请记住：

1. **你已经做了大多数家属做不到的事**——把 4 张化验单、病史时间线整理清楚，问对了问题
2. **不要焦虑，但要主动**——焦虑无益，主动有用
3. **病人本人需要鼓励**——57 岁有慢性病史，容易自责"是我拖累家人"。家人态度温和坚定，不要让她内疚
4. **饮食是表达爱的最直接方式**——花心思给她做易消化又营养的小份餐，比说什么都管用
5. **慢性病程必有起伏**——这次比上次重，但只要根因解决，长期可以好转
6. **照顾者也要照顾自己**——分工、轮换、保留自己睡眠
